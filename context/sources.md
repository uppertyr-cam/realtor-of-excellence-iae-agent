# Sources

## Source Policy

- Official ROE-owned pages are the primary truth source for identity, tone, positioning, services, and public proof points.
- Secondary sources should only supplement gaps and must be labeled as secondary.
- Any inferred brand rule or business priority should be marked as inferred rather than stored as confirmed fact.
